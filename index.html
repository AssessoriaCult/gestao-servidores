Termo de Uso de Dados

Em conformidade com a Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018), informamos que este sistema coleta e trata dados pessoais exclusivamente para fins de gestão interna institucional da Secretaria Municipal de Cultura de Araçatuba.

  • Os dados são utilizados apenas para gestão administrativa dos servidores
  • Não compartilhamos dados com terceiros
  • O acesso é restrito a usuários autorizados pela administração
  • Você pode solicitar a exclusão dos seus dados a qualquer momento
  • Os dados são armazenados de forma segura com controle de acesso
